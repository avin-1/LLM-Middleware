## Типы уроков

### Теоретические уроки
- Объяснение концепций с диаграммами
- Таблицы с ключевыми моментами
- ~200-350 строк контента
- Расчётное время: 30-45 минут

### Уроки с кодом
- Python реализации, которые можно запускать
- Интеграция с фреймворком SENTINEL
- Примеры готовые к копированию
- Расчётное время: 45-60 минут

### Лабораторные упражнения
- Практические задачи
- Проблемы на основе сценариев
- Градуированные уровни сложности
- Расчётное время: 60-120 минут

### Справочные материалы
- Краткие справочники
- Cheat sheets
- Библиотеки паттернов
- Использовать по необходимости

## Настройка окружения

### Минимальные требования

```bash
# Python 3.10+
python --version

# Git
git --version

# Рекомендуется: Виртуальное окружение
python -m venv ai-security-env
source ai-security-env/bin/activate  # Linux/Mac
.\ai-security-env\Scripts\activate   # Windows
```

### Установка SENTINEL

```bash
# Клонируйте репозиторий
git clone https://github.com/DmitrL-dev/AISecurity.git
cd AISecurity/sentinel-community

# Установите зависимости
pip install -r requirements.txt

# Установите пакет SENTINEL
pip install -e .

# Проверьте установку
python -c "from sentinel import configure; print('SENTINEL ready')"
```

### Окружение для лабораторных

Для практических упражнений вам понадобится:

```bash
# Дополнительные зависимости для лабораторных
pip install -r requirements-labs.txt

# Проверьте настройку лабораторных
python -m labs.verify_setup
```

## Как получить максимум от уроков

### Перед каждым уроком
1. Проверьте prerequisites, указанные вверху
2. Подготовьте ваше окружение
3. Просмотрите ключевые выводы предыдущего урока

### Во время каждого урока
1. Внимательно читайте теоретические разделы
2. Набирайте примеры кода (не просто копируйте)
3. Запускайте весь код и наблюдайте за выводом
4. Делайте заметки о ключевых концепциях

### После каждого урока
1. Выполните все упражнения
2. Попробуйте модифицировать примеры кода
3. Просмотрите ключевые выводы
4. Свяжите концепции с предыдущими уроками

## Примеры кода

Каждый урок включает исполняемый код. Вот как их использовать:

### Inline примеры
```rust
// Это отдельный пример
use sentinel_core::engines::SentinelEngine;

let engine = SentinelEngine::new();
let result = engine.analyze("test input");

println!("Detected: {}, Risk: {}", result.detected, result.risk_score);
// Вы можете скопировать и запустить это напрямую
```

### Class-Based примеры
```rust
/// Полная реализация для обучения.
///
/// Скопируйте весь struct в ваш проект для использования.
struct ExampleDetector {
    patterns: Vec<String>,
}

impl ExampleDetector {
    fn new() -> Self {
        Self {
            patterns: Vec::new(),
        }
    }

    fn detect(&self, text: &str) -> DetectionResult {
        // Реализация здесь
        DetectionResult { detected: false }
    }
}

// Использование
let detector = ExampleDetector::new();
let result = detector.detect("test input");
```

### Примеры интеграции с SENTINEL
```rust
use sentinel_core::engines::SentinelEngine;

// Эти примеры показывают реальное использование фреймворка
// Они требуют установленного sentinel_core

let engine = SentinelEngine::new();
let result = engine.analyze("user input to validate");

if result.detected {
    println!("Threat detected! Risk score: {}", result.risk_score);
    println!("Categories: {:?}", result.categories);
    println!("Processing time: {}μs", result.processing_time_us);
}
```

## Отслеживание прогресса

### Самооценка
После каждого модуля проверьте себя:
- Можете ли вы объяснить ключевые концепции?
- Можете ли вы воспроизвести код по памяти?
- Можете ли вы применить эти техники к новым сценариям?

### Сертификация
Пройдите трек сертификации для валидации знаний:
- Письменная оценка (концепции)
- Практическая оценка (лаборатории)
- Задачи на основе сценариев
